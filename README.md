## Application is not production-ready. Development is now deprecated.
I developed this application from September 13 to September 21, 2023 as part of learning the basics of NestJS.

This documentation provides a high-level overview of the BookAI backend project. For detailed information on endpoints, functionalities, and specific code implementations, refer to the source code and comments within the project files.

For any further assistance or inquiries, feel free to reach out to the project maintainers or contributors.
## About BookAI
BookAI is a backend project designed to leverage the GPT Language Model to assist users in finding books based on their favorite authors, books, or genres. This documentation provides an overview of the project structure, installation steps, available scripts, and the technologies used.

## Installation

```bash
$ npm install
```

## Running the app

```bash
docker-compose up -d

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
